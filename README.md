# 🚗 STM32 嵌入式平衡小车 (Balanced Car)

本项目是一个基于 STM32 微控制器的两轮自平衡小车。通过读取六轴姿态传感器的原始数据，结合姿态解算与 PID 控制算法，实现小车的直立平衡、速度控制与转向控制。

## 🛠️ 技术栈与开发环境

- **主控芯片**：STM32F103C8T6
- **底层固件库**：STM32 HAL 库
- **配置工具**：STM32CubeMX
- **构建系统**：CMake + GCC ARM None EABI
- **开发工具**：Visual Studio Code (配合 Clangd 插件)

## 🧩 硬件选型

| 硬件模块 | 型号/规格 | 接口通信方式 | 说明 |
| :--- | :---: | :---: | :--- |
| **主控板** | STM32F103C8T6 最小系统板 | - | 系统核心运算单元 |
| **姿态传感器** | MPU6050 | I2C | 采集加速度与角速度，用于姿态解算 |
| **电机驱动** | TB6612FNG (或 L298N) | PWM + GPIO | 驱动左右直流减速电机 |
| **测速模块** | 霍尔编码器电机 | 定时器正交解码 | 采集车轮转速，用于闭环反馈 |
| **显示模块** | 0.96寸 OLED | I2C / SPI | 实时显示姿态角、速度及调试参数 |
| **避障模块** | SR04 超声波模块 | GPIO (模拟/外部中断) | 前方障碍物检测 |
